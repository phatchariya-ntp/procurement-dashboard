กรอกข้อมูลให้ครบถ้วน ระบบจะคำนวณวันคงเหลือ สถานะ และส่วนต่างราคาให้อัตโนมัติ

เชื่อมต่อ Google Sheets

วาง Web app URL จาก Google Apps Script (ดูวิธีติดตั้งในไฟล์ AppsScript_Code.js ที่ให้ไว้)

1. เปิด Google Sheet ที่จะใช้เก็บข้อมูล 2. Extensions > Apps Script > วางโค้ดจากไฟล์ AppsScript_Code.js 3. Deploy > New deployment > Web app > Execute as: Me, Access: Anyone 4. คัดลอก Web app URL แล้ววางด้านล่างนี้
